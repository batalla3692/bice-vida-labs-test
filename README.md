# BICE VIDA LABS TEST

## Introduction

Developer test for Bice Vida Labs

## Stack

In order to run the project, you must ensure to have the following dependencies installed: 

* Node v13.11
* npm v6.13

## How to start the project

* Download the project and `cd` into his directory.
* Run `npm install` to install all dependencies of `package.json`.
* Start the server with: `sls offline -s dev`.
* Open your favourite browser and navigate to: `http://localhost:3000/dev/api/v1/policies/calculate`.
